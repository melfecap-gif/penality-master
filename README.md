# ⚽ Penalty Strike 2D

Uma experiência imersiva de cobrança de pênaltis desenvolvida para web. Sinta a pressão da torcida, ajuste sua mira e chute para o gol!

## 📝 Sobre o Projeto

O **Penalty Strike 2D** é um simulador de futebol focado na mecânica de precisão. O jogador assume o papel do batedor em um ambiente clássico de 2D, onde a trajetória da bola é definida pelo movimento do mouse.

### Visual e Personagens

O jogo utiliza uma estética 2D limpa com os seguintes elementos:

| Elemento | Descrição Visual |
| :--- | :--- |
| **Jogador** | Humano 2D: Camiseta vermelha, short preto, membros articulados. |
| **Goleiro** | Humano 2D: Camiseta preta, short branco e luvas de proteção. |
| **Ambiente** | Gramado verde vibrante com marcações oficiais (linha de área, marca do pênalti). |
| **Torcida** | Público humano animado nas arquibancadas para aumentar a imersão. |

## 🎮 Mecânicas de Jogo

A jogabilidade é intuitiva, baseada em física de arrastar e soltar:

*   **Mira:** Clique na bola e arraste o mouse para trás ou para os lados para desenhar a trajetória desejada.
*   **Chute:** A bola é disparada no momento em que você solta o botão do mouse.
*   **Objetivo:** Vencer o goleiro e balançar a rede dentro dos limites do gol.

## 🛠️ Tecnologias Utilizadas

*   **Engine:** Antigravity (Desenvolvimento focado em HTML5/Canvas/JavaScript).
*   **Hospedagem:** GitHub Pages.
*   **Versionamento:** GitHub Desktop.
